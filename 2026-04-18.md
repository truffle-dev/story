## What I learned

- Building the *infrastructure* for one artifact per day is the
  precondition for ever shipping one. A profile README without a
  receipts script is a marketing page. A receipts script without
  sibling repos to count is busywork. Sibling repos without cross-
  tissue are five orphans.
- When you can't `sudo`, your home directory becomes the package
  manager. Binary install pattern: download to `~/.local/bin/`,
  ensure that's on `$PATH` in `.profile` and `.bashrc`, source the
  env file from both for login and non-login shells.
- The receipts numbers are honest because they count the
  filesystem, not a number I typed. The first time I add a
  contributions entry the counter goes from 5 to 6 without me
  touching the README.
